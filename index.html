📤 تحميل ملف Excel أو CSV

قم بتحميل ملف Excel أو CSV الذي يحتوي على التذاكر بالأعمدة التالية (كما في النموذج الجديد):

  • Interaction ID (رقم التذكرة)
  • Title (العنوان)
  • Description (الوصف)
  • Solution (الحل)
  • Category (الفئة)
  • Subcategory (الفئة الفرعية)
  • Classification (التصنيف)
  • Type (Service Request / Incident)
  • Priority (الأولوية)
  • Status (حالة التذكرة)

🔍 معايير التحقق المستندة إلى التذاكر الصحيحة

  • العنوان: يجب أن يكون واضحاً ومختصراً (4-5 كلمات) ويحتوي على كلمات مفتاحية مثل: طلب، تعديل، إضافة، حذف، مشكلة، خطأ.
  • الوصف: يجب أن يشرح المشكلة بالتفصيل مع ذكر أرقام الدعاوى/الطلبات.
  • الحل: يجب أن يصف الإجراء المتخذ ويحتوي على كلمات الإنجاز (تم، تمت، انتهى).
  • الفئة والتصنيف: يجب أن تكون متوافقة مع التصنيفات المعروفة (CMS - Cases, CMS - Requests, ...).
  • النوع: Service Request عادةً له Priority 4، Incident له Priority 1-3.
